🛒 Shopping Cart System

A simple Shopping Cart Application built using Java that simulates an online shopping cart where users can add products, manage items, and view their cart.

📌 Features
	•	Add products to cart
	•	Remove items from cart
	•	View cart items
	•	Calculate total price
	•	Simple GUI-based interaction
	•	Organized object-oriented structure

🧩 Components

Product

Represents a product with attributes such as:
	•	Product name
	•	Price
	•	Image

CartItem

Represents an item added to the shopping cart with quantity.

ShoppingCart

Handles core cart functionality:
	•	Add item
	•	Remove item
	•	Display cart
	•	Calculate total price

ShoppingCartApp

Main class used to run the application.

ShoppingCartGUI

Provides a graphical user interface for interacting with the shopping cart.

⚙️ Technologies Used
	•	Java
	•	Object Oriented Programming (OOP)
	•	Java Swing (for GUI)
